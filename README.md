Домашнє завдання за 10 листопада 


Спочатку компіляція з ключем - с : 

g++ -c main.cpp
g++ -c file.cpp

В результаті отримала два об’єктні файли :
 
main.o
file.o
 
Щоб отримати готову виконувану програму я виконала наступне :
 
g++ main.o file.o -o program.out 
./program.out 

________________________________________________________________________

Ключ -с означає “compile only” , тобто компілятор лише читає вхідний код , компілює його, та записує результат в файл .o 
Такий файл називається об’єктним.
Щоб отримати виконувану програму , необхідно злінкувати об’єктний файл з іншими об’єктними файлами.
